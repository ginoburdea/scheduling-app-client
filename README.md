## Installation

```bash
$ git clone https://github.com/ginoburdea/scheduling-app-client.git
$ cd scheduling-app-client
$ npm install
```

## Running the app

```bash
# development mode
$ npm run dev

# build for production
$ npm run build

# preview the production build
$ npm run preview
```

## Testing

```bash
# open cypress to run tests
$ npm run test

# run tests in ci mode
$ npm run test:ci
```
